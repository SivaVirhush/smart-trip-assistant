# Frontend

React app for Incredible Heritage.

## Run

```bash
copy .env.example .env
npm install
npm start
```

## Build

```bash
npm run build
```

## Environment

```bash
REACT_APP_API_URL=http://localhost:5000
REACT_APP_TOMTOM_API_KEY=your_tomtom_key_here
REACT_APP_NEWS_API_KEY=your_newsapi_key_here
```

## Frontend Routes

The app uses these client-side routes:

- `/`
- `/guest`
- `/chat`
- `/chat/new`
- `/chat/:chatId`
- `/features`
- `/calendar`
- `/cultural-maps`
- `/live-traffic`
- `/current-affairs`
